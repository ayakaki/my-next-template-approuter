# my-next-template-approuter

approuter の Next.js アプリケーションを立ち上げる際のテンプレートプロジェクト

## FEATURE

仮置きで下記のパッケージを使用。

- HTTP ライブラリ：Axios
- 状態管理：recoil
- テスティングフレームワーク：Jest
- CSS フレームワーク：Tailwind CSS
